# QR Code Geneator-JS

## Techniques Used
- html
- css
- Java Script

Time Taken: 2 hours

I have Called API from [qrickit.com](https://qrickit.com/qrickit_apps/qrickit_api.php) to generate QR code.  We can pass any text, numbers, mails, url etc.. and get the particular QR code.
